# Pong

A recreation of the classic arcade game **Pong**, built with [Pygame](https://www.pygame.org/).

## About

This project is my take on the classic Pong game, built as a way to keep my Python skills and game development fundamentals sharp.

### Prerequisites

- Python 3.8+
- pip

### Installation

1. Clone the repository
```bash
   git clone https://github.com/Damiano-R06/Pong.git
   cd Pong
```

2. Install dependencies
```bash
   pip install pygame
```

### Running the game

```bash
python src/mainLoop.py
```

## Controls

| Player | Move Up | Move Down |
|--------|---------|-----------|
| Player 1 | `W` | `S` |
| Player 2 | `↑` | `↓` |
